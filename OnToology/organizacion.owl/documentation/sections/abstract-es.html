Síntesis

Este vocabulario se utiliza para la descripción de organizaciones públicas en España, así como de sus estructuras organizativas. Reutiliza los siguientes vocabularios: * W3C Organization Ontology, que se importa en el vocabulario. * SKOS para la representación de listas de códigos sobre tipos de administraciones, niveles de administración, estado de los organismos, etc. * DBpedia ontology, para la representación de un GovenmentalAdministrativeRegion. Este vocabulario se completa con las listas de códigos disponibles en: * http://vocab.linkeddata.es/datosabiertos/kos/sector-publico/organizacion/tipo-entidad * http://vocab.linkeddata.es/datosabiertos/kos/sector-publico/organizacion/estado-entidad

Abstract

This vocabulary can be used for the description of public organisations in Spain, including their organisational structure. It reuses the following vocabularies: * W3C Organization Ontology, which is imported by this vocabulary. * SKOS to represent code lists about types and levels of public administrations, status of organisations, etc. * DBpedia ontology, to represent a GovenmentalAdministrativeRegion. This vocabulary is completed with codes lists available at: * http://vocab.linkeddata.es/datosabiertos/kos/sector-publico/organizacion/tipo-entidad * http://vocab.linkeddata.es/datosabiertos/kos/sector-publico/organizacion/estado-entidad
